# Zip Codes Info Coding Test

###### Prerequisites
Node.js v16 LTS

## Running locally (Node.js)

#### 1. Installing dependencies
```
npm install
```

#### 2. Starting server and web app
```
npm start
```

You might want to run server and app separately, if that's the case you need to run>

```
npm run start:server
npm run start:app
```

## Running unit tests

```
npm run test
```
